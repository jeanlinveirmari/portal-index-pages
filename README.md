# portal-index-pages

本仓库用于归档和发布多个独立的 HTML 页面。这些页面可以用于信息展示、导航索引、项目介绍等场景，不针对任何特定域名或网站。

## 项目简介

`portal-index-pages` 是一个轻量级的 HTML 页面集合仓库。主要目的是将分散的、功能独立的静态页面集中存放，便于版本管理、浏览和部署。每个页面都是自包含的 HTML 文件，可以直接在浏览器中打开查看。

## 目录说明

仓库根目录下存放各独立 HTML 页面文件，建议按功能或主题命名文件，例如：

- `index.html` — 默认入口页面
- `page-example.html` — 示例页面
- 其他页面文件

如有需要，可以创建子目录来分类存放资源文件（如 CSS、JavaScript、图片等），但所有页面主体应保持为独立的 HTML 文件。

## 页面归档说明

- 每个 HTML 文件应尽量自包含，减少对外部资源的依赖（如需引用外部资源，建议使用相对路径或可靠的 CDN 链接）。
- 页面内容应保持简洁、清晰，避免包含敏感信息或违反平台规定的链接。
- 本仓库不提供自动化构建或动态渲染功能，所有页面均为纯静态 HTML。

## 维护说明

- 欢迎通过 Pull Request 提交新的页面或对现有页面进行改进。
- 提交前请确保 HTML 文件格式正确，无明显错误。
- 请勿提交与仓库主题无关的文件，或包含恶意代码、广告追踪等内容。
- 仓库维护者会定期检查页面可用性，但不对第三方内容负责。
